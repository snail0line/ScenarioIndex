# ScenarioIndex

ScenarioIndex는 **파일로 보유한 CardWirth 시나리오를 검색하고 관리할 수 있는 프로그램**입니다.
<br>
ScenarioIndexは、**ローカルに保存されたCardWirthシナリオを検索・管理できるプログラム**です。
